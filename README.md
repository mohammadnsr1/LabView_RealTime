# LabView_RealTime

This repository contains LabVIEW programs for real-time data acquisition and waveform capture using NI hardware, especially PXI/Real-Time and NI Scope based setups.

The included VIs provide:

- Real-time acquisition workflows for continuous measurement on NI PXI real-time targets
- Digital and trigger-based capture programs for controlled waveform collection
- Oscilloscope-style logging tools for viewing and saving multi-channel signals
- Test and example VIs for NI Scope configuration, fetching, and continuous acquisition

This project can help others who need a starting point for building LabVIEW applications for high-speed measurement, triggered recording, data logging, and real-time instrument control.
